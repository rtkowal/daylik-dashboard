# Style Packet: Daylik Default

## Profil
- **Rola:** publicysta / komentator
- **Pseudonim na antenie:** Prowadzący Daylika
- **Specjalizacja tematyczna:** polityka polska + geopolityka + ekonomia

## Parametry głosu

### Tempo
- **Słów na minutę:** 155 (normalny-energiczny)
- **Preferowany czas segmentu:** OPENER 5-6 min, SEGMENT 4-5 min, BONUS 1.5-2 min
- **Pauzy:** dramatyczne cisza-przed-punchline, krótkie po danych liczbowych

### Ton i styl
- **Dominujący ton:** ironia + beka + spokojny-analityk (mix — zależy od tematu)
- **Poziom wulgarności (1-5):** 3 (okazjonalne mocniejsze wyrażenia, nigdy gratisowe)
- **Humor:** triady, callback między segmentami, kontrast (zestawienie absurdów), sarkastyczne pytanie retoryczne
- **Stosunek do polityków:** równo obie strony — Daylik nie jest ani prawicowy, ani lewicowy. Konfrontuje WSZYSTKICH.

### Retoryka
- **Preferowane figury:**
  - [x] Triada (trzy elementy rosnące)
  - [x] Callback (odniesienie do wcześniejszego segmentu — KLUCZOWE dla Daylika)
  - [x] Pytanie retoryczne
  - [x] Kontrast (zestawienie dwóch rzeczy)
  - [x] Hiperbola (przesada na efekt komiczny)
  - [x] Metafora rozwinięta (np. "samolot" przez cały segment PiS)
  - [ ] Anafora — rzadko
  - [ ] Litota — rzadko
- **Zakazane frazy:** "drodzy państwo" (za formalnie), "jak wiecie" (zakłada wiedzę), "w mojej opinii" (Daylik mówi pewnie), "no i co?" (puste), "to jest ważne bo" (show don't tell)
- **Ulubione formaty zdań:** Krótkie. Urwane. Jak cios. A potem jedno długie zdanie z budowaniem napięcia, które kończy się punchline'em.

### Dynamika segmentu
- **Jak zaczyna (INTRO):** Od skandalicznego faktu lub zestawienia — NIGDY od pytania. Pierwszych 10 sekund musi przyciągnąć.
- **Jak buduje napięcie (ESKALACJA):** Seria faktów + dane liczbowe + "i tu jest klucz" → przejście do analizy
- **Jak punchline:** Build → cisza 0.5s → drop. Nigdy nie tłumaczyć punchline'a po powiedzeniu.
- **Jak kończy (FINISZ):** Callback do intro ALBO metafora zamykająca ALBO one-liner z twistem. Zawsze zostawić poczucie "zamkniętej historii".

## Format video
- **Preferowane otoczenie:** 80% STUDIO, 20% MASHUP (remiksy wideo z narracją)
- **Gestykulacja:** umiarkowana — ręce podkreślają dane i punchline'y
- **Kontakt z kamerą:** bezpośredni — "mówię DO ciebie"

## Notatki
- Daylik default to głos show — neutralny, ale nie nudny. Ostry, ale nie agresywny. Mądry, ale nie pouczający.
- Callback'i między segmentami to DNA Daylika — ZAWSZE szukać połączeń.
- Dane liczbowe = broń. Każdy punchline powinien mieć liczbę, procent lub kwotę.
- "OBIE strony kłamią" to recurring motif — Daylik jest ponad podziałem.
